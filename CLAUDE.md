# Claude 작업 가이드라인

이 문서는 Claude가 mobok-callserver 프로젝트에서 작업할 때 따라야 할 가이드라인을 정의합니다.

## 작업 시작 전 필수 사항

### 1. .agents 상위 파일 읽기 (필수)

모든 작업을 시작하기 전에 `.agents` 디렉토리의 상위 레벨 파일들을 **반드시 모두** 읽어야 합니다:

- `.agents/ARCH.md` - 아키텍처 문서
- `.agents/CONVENTION.md` - 코딩 컨벤션 및 규칙

**이 파일들은 작업 시작 전 항상 읽은 상태여야 합니다.**

### 2. 하위 디렉토리 파일 읽기 (필요시)

`.agents` 하위 폴더의 파일들은 필요할 때만 읽습니다:

- 특정 모듈이나 기능과 관련된 작업을 할 때
- 상위 파일에서 참조하는 세부 문서가 있을 때
- 작업 컨텍스트상 추가 정보가 필요할 때

### 3. 작업 전 재확인

작업을 시작하기 직전에 다음을 수행합니다:

1. 작업과 관련된 `.agents` 상위 파일을 한 번 더 읽어 컨텍스트를 확인
2. 필요한 경우 관련 하위 폴더 문서도 재확인
3. 현재 작업이 아키텍처와 컨벤션을 준수하는지 검토

## 작업 워크플로우

```
1. .agents/ARCH.md 읽기 (필수)
2. .agents/CONVENTION.md 읽기 (필수)
3. (필요시) .agents/[하위폴더]/* 관련 파일 읽기
4. 작업 시작 전 필요한 문서 재확인
5. 코드 작업 수행
6. npm run build 실행 및 검증 (필수)
7. 빌드 성공 확인
```

## 작업 완료 전 필수 사항

### npm run build 실행

모든 작업이 끝난 후에는 **반드시** 다음을 수행해야 합니다:

```bash
npm run build
```

**빌드가 성공적으로 완료되어야만 작업이 완료된 것으로 간주합니다.**

### 빌드 실패 시

- 빌드 에러를 확인하고 수정
- 타입 에러, 린트 에러 등을 모두 해결
- 다시 `npm run build` 실행
- 빌드 성공까지 반복

- [ ] 작업 관련 문서 재확인

### 작업 완료 전
- [ ] 코드 작업 완료

- [ ] 모든 에러 해결 완료

## 주의사항

1. **상위 파일 우선**: `.agents` 디렉토리의 상위 레벨 파일(ARCH.md, CONVENTION.md)은 항상 작업 전에 읽어야 합니다.
2. **하위 파일 선택적**: `.agents` 하위 폴더의 파일들은 필요할 때만 읽습니다.
3. **재확인 필수**: 작업 시작 직전에 관련 문서를 한 번 더 확인합니다.
4. **빌드 검증 필수**: 모든 작업 후 `npm run build`로 검증해야 합니다.
5. **빌드 실패 불가**: 빌드가 실패한 상태로 작업을 완료할 수 없습니다.
