# Scripts

이 폴더에는 개발 및 테스트를 위한 유틸리티 스크립트들이 포함되어 있습니다.

## 중앙부처 복지서비스 API 테스트

### 사용법

```bash
# 1페이지 조회 (기본값)
npm run script:test-central-welfare

# 특정 페이지 조회
npm run script:test-central-welfare 2

# 전체 응답 JSON 출력
npm run script:test-central-welfare 1 -- --full
```
